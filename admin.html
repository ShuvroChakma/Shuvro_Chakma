Admin Panel

Add, edit, and remove projects, and update your profile photo. Everything is saved in this browser's localStorage — no server or database needed. Changes appear instantly on the Home, About, and Projects pages.

Profile photo

Used on the About page (and can be reused on the homepage). Stored as an image in your browser.

JPG or PNG, square images work best.

Add a new project

Fields marked with the title and summary are required. Everything else is optional but recommended.

Your projects

Edit or delete existing entries. These are the exact projects shown on the Projects page.

Backup & deployment

Export your projects as a JSON file (useful for backing up or committing to data/projects.json before deploying), import a JSON file, or reset to the sample projects.

Change admin password

Sets a new password for logging into this panel. Important: since this is a static site with no server, the new password is saved only in this browser's storage on this device. It will keep working here, but any other browser or device (or if you clear this browser's data) will still need the original default password until you change it there too.
